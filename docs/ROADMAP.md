# ROADMAP — План развития

## Version 0.1.0 — MVP (1-2 недели)

**Цель**: Работающий граф с Tavily и SQLite

### Фичи
- [ ] Базовый граф из 4 нод (Plan → Search → Analyze → Report)
- [ ] SQLite persistence для state
- [ ] Human-in-the-loop через CLI input()
- [ ] Markdown отчет
- [ ] config.toml для настроек

### Критерий готовности
```bash
deep-research query "transformer architectures for time series"
# → Показывает план
# → Просит подтверждения
# → Ищет
# → Показывает результаты
# → Генерирует report.md
```
